# Changelog

## [0.1.0] - 2026-04-23

Initial release.

- `app-dist login` / `whoami` / `rotate-key` — authentication via macOS Keychain
- `app-dist app create` / `list` / `info` — app management
- `app-dist release` — full build pipeline (archive, sign, notarize, DMG, ZIP, upload)
- `app-dist testers add` / `list` / `notify` — tester management and download links
- `app-dist setup signing` — interactive Apple credential configuration
- `app-dist status` — dashboard overview
